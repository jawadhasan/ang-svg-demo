


# svgeditdemo

Simple usage of canvas lib with angular.

## Description

A simple guide.


### create app

```
ng new demo-app
```
```
ng serve --open
```

```
npm install primeng
npm install primeicons
```

```
ng g c help
```
### Installing

Install dependencies
```
npm ci
```

Run development server
```
npm start

```

Build Project

```
npm run build
```
