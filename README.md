


# svgeditdemo

Simple usage of canvas lib with angular.

## Description

A simple guide.


### create app

```
ng new my-new-app
```
```
ng serve --open
```

```
npm install primeng@13.0.0  
npm install primeicons
```

### Installing

Install dependencies
```
npm ci
```

Run development server
```
npm start

```

Build Project

```
npm run build
```
